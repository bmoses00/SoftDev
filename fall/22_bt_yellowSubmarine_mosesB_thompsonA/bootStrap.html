List groups are a strong component for displaying different kinds of content

This is basic code for a list group

<div class="list-group"> <a href="#" class="list-group-item list-group-item-action">

Notice that you can put a link in the list such that clicking on it routes the user somewhere
This part item in the list group is active

In order to make the list item actiave, just add "list-group-item-action active" in the class
